26 岁穷游日本：一都二府

在一个地方呆久了，会厌烦，会顺其自然；想一个地方很久了，会憧憬，会欣然向往；去一个地方要走了，会遗憾，会念念不忘。——《花凋缘尽殇》

上半年的时候准备去台湾，由于某些原因没去成；夏季的某一天，同事告诉我香港快运正在促销，然后就此成行…

# 前期准备

# 机票

香港快运（HK Express (opens new window)）促销，往返机票低于东京到京都的新干线；缺点就是需要提前大半年确定日期，结果行程定在了日本赏枫季之前，完美错过！

# 酒店

同事从马蜂窝上挑选的，性价比不错（原来大数据杀熟离我如此之近）；个人对食宿的要求向来不高，下单：日本宫殿酒店、第一舱京都乌丸、大阪新今宫酒店。

# 签证

一事不烦二主，签证同样在马蜂窝找的旅行社；金卡免流水（需要在职证明，是深户的话可以极简办理），广州送签；脑残买了拒签险，完全可以省下来买电话卡。

# 网络

起先打算租 WiFi 蛋，后来感觉每天出门得多带一设备很麻烦，作罢；（事实证明确实不方便，两手空空出门玩才是最佳状态）经过查询对比，选中了亿点连接全球上网卡：每天 500M 4G 流量、用完无限流量（降速），而日本行程 4G 流量从未用完过，还能免国际漫游费（收取呼叫转移的市话费）接听国内电话；顺便提下：随卡送的折扣券（一堆）完全用不上，日本商场、药妆店对国外游客都是直接出示护照满额免税、打折的。

# 语言

以我之前去泰国 (opens new window)的经验，跛脚英语外加肢体语言足够应天日常使用；实在不行，掏出手机打开谷歌翻译，选中对话，完全可以替代翻译蛋；当然，有语言天赋、兴趣可以学一些基本日语，Japanglish 听得直接懵逼！

# 日常用语

すみません Su mi ma sen：通用句式，常用于寒暄、开头，意思有：对不起、打扰了、请问，类似 Excuse me

# 软件

  • 香港快运航空 (opens new window)：办理网上值机
  • Google 翻译 (opens new window)：功能强大的翻译软件，完全可以代替翻译蛋
  • Google 地图 (opens new window)：查路线、出行方案
  • 东京地铁游乘客指南 (opens new window)：东京地铁查询，可以显示当前到站
  • 鉄道の達人 (opens new window)：京都 Bus、地铁查询；
  • Suica Reader (opens new window)：读取日本公交卡余额。

# 其他

直接手机银行客户端购汇（日元），然后预约取款；淘宝买的安联旅游保险…

# 行程开始

如果不参考行程，想提前获取日本的相关信息；可以跳过本段，直接看注意事项！

# Day 1 - 香港

下班后直接从公司出发去深圳湾口岸，按“持第三国机票与签证陆路赴港乘机”过关，顺便吐槽下：同一个国家还要填入境单；乘坐 B3X 到屯门市中心（11 HKD），然后转 E33 直接到香港国际机场。

# Day 2 -东京

飞行 4 个小时左右到日本羽田国际机场（离东京市区较近，成田机场在千叶县），机场上空手机就接收到网络信号并提示切换时区；过关以后可以拿护照购买东京地铁通票 - Tokyo Subway Ticket (opens new window)，然后坐京急线（410 円）到市区，再根据酒店位置进行换乘；


日本酒店貌似都是 15:00 之后才能办理入住，好在可以存放东西；

# 交通

  • 日本的地铁按区间收费的，需要自己查看线路图上的价格然后去自助售票机上选对应的金额完成买票。买一张 Suica 或 ICOCA 交通卡是很有必要的（便利店可以使用，同深圳通）
  • 日本的地铁、新干线（类似于火车高铁）、特急电车异常的准时，在我待过的地方、时间从没遇到晚点。

# 通票

需要注意的是，针对游客和本地人有不同的票种：

  • 东京：Tokyo Subway Ticket (opens new window)
  • 京都：地铁、巴士一日券 (opens new window)
  • 大阪：大阪周游卡 (opens new window) PS. Tokyo Subway Ticket 除了 JR 及私铁以外随意坐，再买张公交卡备用（由于行程固定、景点少，最后在大阪才买了卡）

# 生活

  • 日本是靠左行驶的国家，与中国大陆相反。
  • 日本到处都是饮料自动售货机，几乎隔几百米就一台。

# 人文

  • 车站、地铁、新干线上没什么人大声说话，更别说吃东西、打电话，这方面中国就呵呵；不是说中国人的素质低，日本地铁上的中国人就没在国内那么放肆了，关键还是中国没有此类的环境
  • 日本人酷爱点头，见面打招呼就点头；以前是听说，现在总算亲眼见过。
  • 日本人真心很热情，特别是老年人：在京都搭巴士，露出为难状就会有同行的老爷爷、老奶奶上前搭讪（虽说语言不通，我不乏善意的猜测为是否需要帮忙之类的话语）
  • 日本人很“蠢”萌，当你找路人询问时，对方甚至会直接送你过去，不知道的时候也会一直尝试提供帮助到你说可以为止

# 影城

大阪环球影城不管是否是周末到处是人，快票会快一些，但是买快票的人也多。

纸上得来终觉浅，绝知此事要躬行；世界那么大，生命太短暂，想多去看看…
