Linux 系统搭建 Node.js 环境

# 下载安装

去官网下载和自己系统匹配的文件： https://nodejs.org/dist/ (opens new window) 英文网址：https://nodejs.org/en/download/ (opens new window) 中文网址：http://nodejs.cn/download/ (opens new window) 下载下来的 tar 文件上传到服务器并且解压，然后通过建立软连接变为全局；

cd /usr/local/
wget -O node-v14.18.1-linux-x64.tar.gz https://nodejs.org/download/release/v14.18.1/node-v14.18.1-linux-x64.tar.gz
tar -zxvf node-v14.18.1-linux-x64.tar.gz
ln -s node-v14.18.1-linux-x64 nodejs
