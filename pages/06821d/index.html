本地开发：泛（内网）域名解析

# 简介

https://nip.io (opens new window) 是免费的 DNS 泛域名解析服务，将符合下列格式的域名解析为对应的 ip；可用来作为本地开发的解析服务，代替配置本地 hosts 文件的步骤。

# 用法

可以使用“IP &.nip.io”和“字段 & IP &.nip.io”的使用方式，其中：

# IP

  • 点分十进制
  • 十进制（用“-”连接）
  • 换算成十六进制

IP 地址进制转换 - https://tool.520101.com/wangluo/jinzhizhuanhuan (opens new window)

127.0.0.1.nip.io
127-0-0-1.nip.io
7f000001.nip.io
1
2
3

# 字段

完全自定义，允许多层级

app.127.0.0.1.nip.io
app-127-0-0-1.nip.io
app.7f000001.nip.io
app-7f000001.nip.io
1
2
3
4
