asar 文件解包、修改再打包

Electron 程序打包的会在 resources 目录生成一个 app.asar 文件，大部分的资源页面文件都在这边；需要 Node.js 环境

以 XMind 为例（已失效）

#安装工具
npm install -g asar
#解包
asar extract app.asar <目录>
#打包
asar pack <目录> app.asar
