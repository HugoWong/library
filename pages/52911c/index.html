# 镜像准备

Zabbix 官方镜像 (opens new window)

# MySQL

docker pull mysql:8.0
1

# Server

Zabbix-Server 镜像分两种：支持 MySQL 数据库的 zabbix-server-mysql (opens new window) 和支持 PostgreSQL 数据库的 zabbix-server-pgsql (opens new window)

docker pull zabbix/zabbix-server-mysql:latest
1

# Web

官方共有五种，习惯性使用 Nginx，所以选择 zabbix-web-nginx-mysql (opens new window)

docker pull zabbix/zabbix-web-nginx-mysql:latest
1

# Java Gateway

选配，Zabbix 默认不支持 Java，需要安装 zabbix-java-gateway (opens new window) 来监控 JVM/Tomcat 的性能，按需选配

 docker pull zabbix/zabbix-java-gateway:latest
1

docker run --name zabbix-mysql -dit \
      -p 13306:3306 \
      --restart=always \
      --network zabbix_net \
      -e MYSQL_DATABASE="zabbix" \
      -e MYSQL_USER="zabbix" \
      -e MYSQL_PASSWORD="zabbix1@#$" \
      -e MYSQL_ROOT_PASSWORD="root_1@#$" \
      --character-set-server=utf8 --collation-server=utf8_bin
      -v /data/zabbix/mysql:/var/lib/mysql \
      -v /etc/localtime:/etc/localtime \
