极路由 1S 刷第三方固件

之前一直将树莓派 3B（OpenWrt）作当主路由器使用，华为 WS5200 做 AP，可无奈树莓派只有一个网卡，又不想旁路，正好老家升级千兆网络，远古的极路由 1S 就空出来了，刚好拿来用

本文适用于刷所有的第三方固件

# 前期准备

# Breed

从 https://breed.hackpascal.net/ (opens new window) 网站下载极路由 1S（hc5661 没有 a）固件 mt7620-hiwifi-hc5761.bin (opens new window)

# OpenWrt

官方存档 - https://archive.openwrt.org/releases/ (opens new window)

21.02.3 (opens new window)

22.03.0-rc1 (opens new window)

# 刷机流程

# SSH 连接

需要开启开发者模式（厂家已经倒闭，现存没有开启开发者模式的设备应该不多），直接 SSH 连接，端口号是 1022

用户名：root，密码是路由器密码

# 刷 u-boot

PS.请认准设备型号，刷错变砖要用编程器，很麻烦

路由器没有开启 SFTP 服务，用 WinSCP 通过 SCP 协议建立一个会话

然后进入想要存放镜像的目录，如：/tmp，将镜像拖拽进去即可

cd /tmp
mtd write breed-mt7620-hiwifi-hc5761.bin u-boot
1
2

断开极路由电源，按住 reset，再通电直至路由器指示灯全闪后（5 秒左右），浏览器访问 192.168.1.1 (opens new window) 进入 breed Web 控制台，备份完整固件和 eeprom，刷入 OpenWrt 即可

# 刷 OpenWrt

进入 breed 界面，切换到“固件更新”，勾选“固件”，选择下载好的镜像，点击上传

更新

然后完成

# OpenWrt

刷入成功以后，OpenWrt 管理地址 192.168.1.1，账户 root，密码等待设置

# Padavan

  • hiboy (opens new window)，搜索 ji1S 或者 7620，亲测三个 trx 文件都可以用，强迫症，最后用了 ji1S 版本
  • chongshengB (opens new window)，使用 Github Actions 自行编译 (opens new window)

刷入成功以后，Padavan 管理地址 192.168.2.1，账户密码都是 admin，先从“系统管理”->“配置管理”里恢复nvram和/etc/storage
