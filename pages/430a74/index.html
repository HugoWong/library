# 简介

深度操作系统（deepin (opens new window)）是一款致力于为全球用户提供美观易用、安全稳定服务的 Linux 发行版，同时也一直是排名最高的来自中国团队研发的 Linux 发行版。点击了解 deepin 国际排名 (opens new window)

Deepin 20.5 (opens new window) 版本已经发布，下文使用 VirtualBox (opens new window) 进行安装体验

# 系统设置

个人习惯：电脑、手机登电子数码产品，开机后的第一件事就是摸清楚可以做哪些系统设置；挨个点个遍，该设备基本上就熟悉了

# 增强功能

从菜单栏依次点击“设备”->“安装增强功能”，Deepin 会加载一个 ISO 文件，但是不会自动运行

需要手动打开“文件管理器”，找到“VBoxLinuxAdditions.run”文件，拖拽到终端用sudo命令执行

sudo '/media/Wei/VBox_GAs_6.1.32/VBoxLinuxAdditions.run'
1

安装完成重启，可以从菜单栏“设备”菜单中按需设置共享粘贴板、拖放：全部双向
